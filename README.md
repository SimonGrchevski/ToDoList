# toToList

> This project is simple minimalist to-do-list.
### Features:

  - Add new Task
  - Remove Task
  - Reorder tasks
  - Delete task
  - Clear completed tasks

## Getting Started


### Prerequisites
  **webpack**,
  **HtmlWebpackPlugin**,
  **style-loader css-loader**,
  **webpack-dev-server**

## Setup

#### $git clone https://github.com/SimonGrchevski/ToDoList.git
#### $cd to-do-list
#### $npm install
#### $npm run build
#### $npm start
